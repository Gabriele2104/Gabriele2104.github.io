Progetto Stelle!

Questo è stato il primo progetto che abbiamo realizzato e lo abbiamo chiamato Progetto Stelle. Si è chiamato così perché il nostro insegnate essendo un appassionato di astronomia ha deciso di farci fare un sito web che parlasse delle stelle, costellazioni ecc... Essendo il primo progetto realizzato siamo partiti con le seguenti tecnologie: HTML, CSS e JAVASCRIPT. Lo abbiamo reso responsivo che è la parte più importante per un sito web, webApp ecc... per i dispositivi mobili. Per farlo abbiamo utilizzato le Media Query con flex-box. Invece per cambiare lo stato al bottone del menù abbiamo usato CSS, invece JAVASCRIPT lo abbiamo utilizzato per associare le classi in modo che quando il menù è a forma di Hamburger il menù non è visibile, mentre al click diventa a forma di croce e il menù diventa visibile.
